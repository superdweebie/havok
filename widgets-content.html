Using Widgets

Individual or complied

As with all havok modules, widgets are all comply with the AMD specification. They may be loaded individually, built into your own compiled layer with the dojo build tool (or another build tool), or used from the compiled havok.js distributable.

Declataive use

All widgets can be defined declatively as valid HTML and then instantated with the havok parser.

To define declartively, use the havok custom tags, the havok/parser/complete! plugin:

  1. <script>
  2.     require(['havok/parser/complete!'], function(){//do something here after widgets parsed})
  3. </script>
  4. <w-button>One widget button</w-button>

Programatic use

To define programatically, first create an instance, then add to the dom, and call `startup`:

  1. <script>
  2.     require(['havok/widget/Button', 'dojo/dom', 'dojo/domReady!'], function(Button, dom){
  3.         var button = new Button({text: 'One widget button'});
  4.         dom.byId('button').appendChild(button.domNode);
  5.         button.startup();
  6.     })
  7. </script>
  8. <div id="button"></div>

Leveraging the power of dojo

Widgets are built on dojo's dijit framework, and so use the standard dijit lifecycle, templating, etc.

Programatic Example

To create a havok/widget/DropdownToggle programatically, pass an instance of havok/widget/_BaseDropdown to the dropdown property, and the markup to create the button to the innerHTML property.

  1. <div id="dropdownPosition1"></div>
  2. <script type="text/javascript">
  3.     require(['dojo/dom', 'dojo/dom-construct', 'havok/widget/Dropdown', 'havok/widget/DropdownToggle', 'havok/ready!'],
  4.     function(dom, domConstruct, Dropdown, DropdownToggle){
  5.  
  6.         /*create the dropdown widget*/
  7.         var dropdown = new Dropdown;
  8.         dropdown.addItem('<a href="">Action</a>');
  9.         dropdown.addItem('<a href="">Another Action</a>');
  10.         dropdown.addItem('<a href="">Something else here</a>');
  11.         dropdown.addItem('<hr />');
  12.         dropdown.addItem('<a href="">Separated Link</a>');
  13.         dropdown.startup();
  14.  
  15.         /*create and start the dropdown toggle*/
  16.         var dropdownToggle = new DropdownToggle({dropdown: dropdown, innerHTML: '<button class="btn">Click me</button>'});
  17.         dom.byId('dropdownPosition1').appendChild(dropdownToggle.domNode);
  18.         dropdownToggle.startup();
  19.     })
  20. </script>
