          dijit/Viewport

          Utility singleton to watch for viewport resizes, avoiding duplicate notifications which can lead to infinite loops.

          Usage: Viewport.on("resize", myCallback).

          myCallback() is called without arguments in case it&#39;s _WidgetBase.resize(), which would interpret the argument as the size to make the widget.
