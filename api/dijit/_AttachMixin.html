          dijit/_AttachMixin

          Mixin for widgets to attach to dom nodes and setup events via convenient data-dojo-attach-point and data-dojo-attach-event DOM attributes.

          Superclass of _TemplatedMixin, and can also be used standalone when templates are pre-rendered on the server.

          Does not [yet] handle widgets like ContentPane with this.containerNode set. It should skip scanning for data-dojo-attach-point and data-dojo-attach-event inside this.containerNode, but it doesn&#39;t.

          Usage

          _AttachMixin is a mixin. It is not intended to be used alone.

          To extend a class with _AttachMixin use declare. Eg:

          1. require(['dojo/_base/declare', 'my/parent/class', 'dijit/_AttachMixin'], function(declare, MyParent, _AttachMixin){
          2.     var MyChildClass = declare([MyParent, _AttachMixin], function(){
          3.         //other child extensions here
          4.     });
          5.  
          6.     var child = new MyChildClass; //create an instance of MyChildClass
          7. });

          attachScope

          Defined by dijit/_AttachMixin

          Types: Object

          Tagged: public

          Object to which attach points and events will be scoped. Defaults to &#39;this&#39;.

          searchContainerNode

          searchContainerNode

          Defined by dijit/_AttachMixin

          Types: Boolean

          Tagged: protected

          Search descendants of this.containerNode for data-dojo-attach-point and data-dojo-attach-event. Should generally be left false (the default value) both for performance and to avoid failures when this.containerNode holds other _AttachMixin instances with their own attach points and events.

          _attach

          _attach(node, type, func) - private

          Defined by dijit/_AttachMixin

          Roughly corresponding to dojo/on, this is the default function for processing a data-dojo-attach-event. Meant to attach to DOMNodes, not to widgets.

          Parameter Type Usage Summary
          node DOMNode required

          The node to setup a listener on.
          type String required

          Event name like "click".
          func undefined required

          Returns: undefined

          _attachTemplateNodes

          _attachTemplateNodes(rootNode) - private

          Defined by dijit/_AttachMixin

          Iterate through the dom nodes and attach functions and nodes accordingly.

          Map widget properties and functions to the handlers specified in the dom node and it&#39;s descendants. This function iterates over all nodes and looks for these properties:

          • dojoAttachPoint/data-dojo-attach-point
          • dojoAttachEvent/data-dojo-attach-event
          Parameter Type Usage Summary
          rootNode DomNode required

          The node to search for properties. All descendants will be searched.
