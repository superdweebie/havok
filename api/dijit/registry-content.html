          findWidgets

          findWidgets(root, skipNode)

          Defined by dijit/registry

          Search subtree under root returning widgets found. Doesn&#39;t search for nested widgets (ie, widgets inside other widgets).

          Parameter Type Usage Summary
          root DOMNode required

          Node to search under.
          skipNode DOMNode required

          If specified, don&#39;t search beneath this node (usually containerNode).

          Returns: Array

          getEnclosingWidget

          getEnclosingWidget(node)

          Defined by dijit/registry

          Returns the widget whose DOM tree contains the specified DOMNode, or null if the node is not contained within the DOM tree of any widget

          Parameter Type Usage Summary
          node DOMNode required

          Returns: undefined | null

          getUniqueId
