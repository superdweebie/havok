                      dojo/dom-construct

                      Methods

                      All methods are filtered from view.

                      create

                      create(tag, attrs, refNode, pos)

                      Defined by dojo/dom-construct

                      Create an element, allowing for optional attribute decoration and placement.

                      A DOM Element creation function. A shorthand method for creating a node or a fragment, and allowing for a convenient optional attribute setting step, as well as an optional DOM placement reference.

                      Attributes are set by passing the optional object through dojo.setAttr. See dojo.setAttr for noted caveats and nuances, and API if applicable.

                      Placement is done via dojo.place, assuming the new node to be the action node, passing along the optional reference node and position.

                      Parameter Type Usage Summary
                      tag DOMNode | String required

                      A string of the element to create (eg: "div", "a", "p", "li", "script", "br"), or an existing DOM node to process.
                      attrs Object required

                      An object-hash of attributes to set on the newly created node. Can be null, if you don&#39;t want to set any attributes/styles. See: dojo.setAttr for a description of available attributes.
                      refNode DOMNode | String optional

                      Optional reference node. Used by dojo.place to place the newly created node somewhere in the dom relative to refNode. Can be a DomNode reference or String ID of a node.
                      pos String optional

                      Optional positional reference. Defaults to "last" by way of dojo.place, though can be set to "first","after","before","last", "replace" or "only" to further control the placement of the new node relative to the refNode. &#39;refNode&#39; is required if a &#39;pos&#39; is specified.

                      Returns: undefined

                      Examples
                      1. Create a DIV:
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     var n = domConstruct.create("div");
                      4. });
                      1. Create a DIV with content:
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     var n = domConstruct.create("div", { innerHTML:"<p>hi</p>" });
                      4. });
                      1. Place a new DIV in the BODY, with no attributes set
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     var n = domConstruct.create("div", null, dojo.body());
                      4. });
                      1. Create an UL, and populate it with LI&#39;s. Place the list as the first-child of a
                      2. node with id="someId":
                      3. require(["dojo/dom-construct", "dojo/_base/array"],
                      4. function(domConstruct, arrayUtil){
                      5.     var ul = domConstruct.create("ul", null, "someId", "first");
                      6.     var items = ["one", "two", "three", "four"];
                      7.     arrayUtil.forEach(items, function(data){
                      8.         domConstruct.create("li", { innerHTML: data }, ul);
                      9.     });
                      10. });
                      1. Create an anchor, with an href. Place in BODY:
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     domConstruct.create("a", { href:"foo.html", title:"Goto FOO!" }, dojo.body());
                      4. });

                      place

                      place(node, refNode, position)

                      Defined by dojo/dom-construct

                      Attempt to insert node into the DOM, choosing from various positioning options. Returns the first argument resolved to a DOM node.

                      Parameter Type Usage Summary
                      node DOMNode | String required

                      id or node reference, or HTML fragment starting with "<" to place relative to refNode
                      refNode DOMNode | String required

                      id or node reference to use as basis for placement
                      position String | Number optional

                      string noting the position of node relative to refNode or a number indicating the location in the childNodes collection of refNode. Accepted string values are:

                      • before
                      • after
                      • replace
                      • only
                      • first
                      • last

                      "first" and "last" indicate positions as children of refNode, "replace" replaces refNode, "only" replaces all children. position defaults to "last" if not specified

                      Returns: DOMNode | undefined

                      Examples
                      1. Place a node by string id as the last child of another node by string id:
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     domConstruct.place("someNode", "anotherNode");
                      4. });
                      1. Place a node by string id before another node by string id
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     domConstruct.place("someNode", "anotherNode", "before");
                      4. });
                      1. Create a Node, and place it in the body element (last child):
                      2. require(["dojo/dom-construct", "dojo/_base/window"
                      3. ], function(domConstruct, win){
                      4.     domConstruct.place("<div></div>", win.body());
                      5. });
                      1. Put a new LI as the first child of a list by id:
                      2. require(["dojo/dom-construct"], function(domConstruct){
                      3.     domConstruct.place("<li></li>", "someUl", "first");
                      4. });
