                      dojo/number.__FormatOptions

                      Usage

                      1. require(['dojo/number.__FormatOptions'], function(number.__FormatOptions){
                      2.     var foo = new number.__FormatOptions();
                      3.     //do something with foo
                      4. });

                      Properties

                      All properties are filtered from view.

                      fractional

                      fractional

                      Defined by dojo/number

                      Types: Boolean

                      If false, show no decimal places, overriding places and pattern settings.

                      locale

                      locale

                      Defined by dojo/number

                      Types: String

                      override the locale used to determine formatting rules

                      pattern

                      pattern

                      Defined by dojo/number

                      Types: String

                      override formatting patternwith this string. Default value is based on locale. Overriding this property will defeat localization. Literal characters in patterns are not supported.

                      places

                      places

                      Defined by dojo/number

                      Types: Number

                      fixed number of decimal places to show. This overrides any information in the provided pattern.

                      round

                      round

                      Defined by dojo/number

                      Types: Number

                      5 rounds to nearest .5; 0 rounds to nearest whole (default). -1 means do not round.

                      type

                      type

                      Defined by dojo/number

                      Types: String

                      choose a format type based on the locale from the following: decimal, scientific (not yet supported), percent, currency. decimal by default.
