                      dojo/dom-style

                      This module defines the core dojo DOM style API.

                      Methods

                      All methods are filtered from view.

                      get

                      get(node, name)

                      Defined by dojo/dom-style

                      Accesses styles on a node.

                      Getting the style value uses the computed style for the node, so the value will be a calculated value, not just the immediate node.style value. Also when getting values, use specific style names, like "borderBottomWidth" instead of "border" since compound values like "border" are not necessarily reflected as expected. If you want to get node dimensions, use dojo/dom-geometry.getMarginBox(),dojo/dom-geometry.getContentBox() or dojo/dom-geometry.getPosition().

                      Parameter Type Usage Summary
                      node DOMNode | String required

                      id or reference to node to get style for
                      name String optional

                      the style property to get

                      Returns: undefined

                      Examples
                      1. Passing only an ID or node returns the computed style object of
                      2. the node:
                      3. require(["dojo/dom-style", "dojo/dom"], function(domStyle, dom){
                      4.     domStyle.get("thinger");
                      5. });
                      1. Passing a node and a style property returns the current
                      2. normalized, computed value for that property:
                      3. require(["dojo/dom-style", "dojo/dom"], function(domStyle, dom){
                      4.     domStyle.get("thinger", "opacity"); // 1 by default
                      5. });
                      getComputedStyle

                      getComputedStyle(node)

                      Defined by dojo/dom-style

                      Returns a "computed style" object.

                      Gets a "computed style" object which can be used to gather information about the current state of the rendered node.

                      Note that this may behave differently on different browsers. Values may have different formats and value encodings across browsers.

                      Note also that this method is expensive. Wherever possible, reuse the returned object.

                      Use the dojo/dom-style.get() method for more consistent (pixelized) return values.

                      Parameter Type Usage Summary
                      node DOMNode required

                      A reference to a DOM node. Does NOT support taking an ID string for speed reasons.
                      Examples
                      1. require(["dojo/dom-style", "dojo/dom"], function(domStyle, dom){
                      2.     domStyle.getComputedStyle(dom.byId(&#39;foo&#39;)).borderWidth;
                      3. });
                      1. Reusing the returned object, avoiding multiple lookups:
                      2. require(["dojo/dom-style", "dojo/dom"], function(domStyle, dom){
                      3.     var cs = domStyle.getComputedStyle(dom.byId("someNode"));
                      4.     var w = cs.width, h = cs.height;
                      5. });
                      set

                      set(node, name, value)

                      Defined by dojo/dom-style

                      Sets styles on a node.

                      Parameter Type Usage Summary
                      node DOMNode | String required

                      id or reference to node to set style for
                      name String | Object required

                      the style property to set in DOM-accessor format ("borderWidth", not "border-width") or an object with key/value pairs suitable for setting each property.
                      value String optional

                      If passed, sets value on the node for style, handling cross-browser concerns. When setting a pixel value, be sure to include "px" in the value. For instance, top: "200px". Otherwise, in some cases, some browsers will not apply the style.

                      Returns: String | undefined

                      Examples
                      1. Passing a node, a style property, and a value changes the
                      2. current display of the node and returns the new computed value
                      3. require(["dojo/dom-style"], function(domStyle){
                      4.     domStyle.set("thinger", "opacity", 0.5); // == 0.5
                      5. });
                      1. Passing a node, an object-style style property sets each of the values in turn and returns the computed style object of the node:
                      2. require(["dojo/dom-style"], function(domStyle){
                      3.     domStyle.set("thinger", {
                      4.         "opacity": 0.5,
                      5.         "border": "3px solid black",
                      6.         "height": "300px"
                      7.     });
                      8. });
                      1. When the CSS style property is hyphenated, the JavaScript property is camelCased.
                      2. font-size becomes fontSize, and so on.
                      3. require(["dojo/dom-style", "dojo/dom"], function(domStyle, dom){
                      4.     domStyle.set("thinger",{
                      5.         fontSize:"14pt",
                      6.         letterSpacing:"1.2em"
                      7.     });
                      8. });
                      1. dojo/NodeList implements .style() using the same syntax, omitting the "node" parameter, calling
                      2. dojo/dom-style.get() on every element of the list. See: dojo/query and dojo/NodeList
                      3. require(["dojo/dom-style", "dojo/query", "dojo/NodeList-dom"],
                      4. function(domStyle, query){
                      5.     query(".someClassName").style("visibility","hidden");
                      6.     // or
                      7.     query("#baz > div").style({
                      8.         opacity:0.75,
                      9.         fontSize:"13pt"
                      10.     });
                      11. });
