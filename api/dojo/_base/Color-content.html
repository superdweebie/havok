                    constructor

                    constructor(color)

                    Defined by dojo/_base/Color

                    Takes a named string, hex string, array of rgb or rgba values, an object with r, g, b, and a properties, or another Color object and creates a new Color instance to work from.

                    Parameter Type Usage Summary
                    color Array | String | Object required
                    Examples
                    1. Work with a Color instance:
                    2. require(["dojo/_base/color"], function(Color){
                    3.     var c = new Color();
                    4.     c.setColor([0,0,0]); // black
                    5.     var hex = c.toHex(); // #000000
                    6. });
                    1. Work with a node&#39;s color:
                    3.   require(["dojo/_base/color", "dojo/dom-style"], function(Color, domStyle){
                    4.       var color = domStyle("someNode", "backgroundColor");
                    5.       var n = new Color(color);
                    6.       // adjust the color some
                    7.       n.r *= .5;
                    8.       console.log(n.toString()); // rgb(128, 255, 255);
                    9.   });

                    fromArray(a, obj)

                    Defined by dojo/_base/Color

                    Builds a Color from a 3 or 4 element array, mapping each element in sequence to the rgb(a) values of the color.

                    Parameter Type Usage Summary
                    a Array required
                    obj havok/widget/Color optional

                    Returns: any

                    Examples
                    1. require(["dojo/_base/color"], function(Color){

                    3. });
                    fromHex

                    fromHex(color, obj)

                    Defined by dojo/_base/Color

                    Converts a hex string with a &#39;#&#39; prefix to a color object. Supports 12-bit #rgb shorthand. Optionally accepts aColor object to update with the parsed value.

                    Parameter Type Usage Summary
                    color String required
                    obj havok/widget/Color optional

                    Returns: any | null | string

                    Examples
                    1. require(["dojo/_base/color"], function(Color){
                    2.     var thing = new Color().fromHex("#ededed"); // grey, longhand
                    3.     var thing2 = new Color().fromHex("#000"); // black, shorthand
                    4. });

                    fromString

                    fromString(str, obj)

                    Defined by dojo/_base/Color

                    Parses str for a color value. Accepts hex, rgb, and rgba style color values.

                    Acceptable input values for str may include arrays of any form accepted by dojo.colorFromArray, hex strings such as "#aaaaaa", or rgb or rgba strings such as "rgb(133, 200, 16)" or "rgba(10, 10, 10, 50)"

                    Parameter Type Usage Summary
                    str String required
                    obj havok/widget/Color optional

                    Returns: any
