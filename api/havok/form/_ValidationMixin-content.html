                            havok/form/_ValidationMixin

                            Inherits from havok/form/_ValidationStyleMixin , mixins havok/form/_ValidationMessagesMixin

                            Mixin that add validation to form inputs

                            Usage

                            _ValidationMixin is a mixin. It is not intended to be used alone.

                            To extend a class with _ValidationMixin use declare. Eg:

                            1. require(['dojo/_base/declare', 'my/parent/class', 'havok/form/_ValidationMixin'], function(declare, MyParent, _ValidationMixin){
                            2.     var MyChildClass = declare([MyParent, _ValidationMixin], function(){
                            3.         //other child extensions here
                            4.     });
                            5.  
                            6.     var child = new MyChildClass; //create an instance of MyChildClass
                            7. });

                            _setValidatorAttr

                            _setValidatorAttr(value) - private

                            Defined by havok/form/_ValidationMixin

                            Will set the validator. May be one of three types:

                            Instance of mystique/Base - the validator property is set equal to this instance.

                            Array - if an array, it is assumed to be an array of validators, or validator definitions. The array will be passed to validatorFactory.create(). The validator property will be set to the returned instance of Validator/Group

                            Object - an an object, it is assumbed to be a validator definition. The definition will be passed to havok/validator/factory.create(). The validator property will be set to the returned instance of mytique/Base

                            Parameter Type Usage Summary
                            value String | String[] | Object | mystique/Base required
