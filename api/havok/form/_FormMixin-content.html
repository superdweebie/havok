                            havok/form/_FormMixin

                            Inherits from dijit/form/_FormMixin

                            Mixin for form like widgets

                            Usage

                            _FormMixin is a mixin. It is not intended to be used alone.

                            To extend a class with _FormMixin use declare. Eg:

                            1. require(['dojo/_base/declare', 'my/parent/class', 'havok/form/_FormMixin'], function(declare, MyParent, _FormMixin){
                            2.     var MyChildClass = declare([MyParent, _FormMixin], function(){
                            3.         //other child extensions here
                            4.     });
                            5. Â 
                            6.     var child = new MyChildClass; //create an instance of MyChildClass
                            7. });

                            Properties

                            All properties are filtered from view.

                            postActivity

                            postActivity

                            Defined by havok/form/_FormMixin

                            Types: Boolean

                            Has the form had user interaction?

                            state

                            state

                            Defined by dijit/form/_FormMixin

                            Types: String

                            Tagged: readonly

                            Will be "Error" if one or more of the child widgets has an invalid value, "Incomplete" if not all of the required child widgets are filled in. Otherwise, "", which indicates that the form is ready to be submitted.
