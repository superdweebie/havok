                            havok/form/_RequiredStarMixin

                            A mixin that facilities marking form inputs as required.

                            Adds the supplied template as an appendage to the label if validator is the same as the requiredValidatorDef

                            Usage

                            _RequiredStarMixin is a mixin. It is not intended to be used alone.

                            To extend a class with _RequiredStarMixin use declare. Eg:

                            1. require(['dojo/_base/declare', 'my/parent/class', 'havok/form/_RequiredStarMixin'], function(declare, MyParent, _RequiredStarMixin){
                            2.     var MyChildClass = declare([MyParent, _RequiredStarMixin], function(){
                            3.         //other child extensions here
                            4.     });
                            5.  
                            6.     var child = new MyChildClass; //create an instance of MyChildClass
                            7. });
