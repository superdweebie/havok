                            havok/di/Di

                            An object used for dependency injection.

                            The config property may be populated with a config object that defines what to inject.

                            Usage

                            1. require(['havok/di/Di'], function(Di){
                            2.     var foo = new Di(config);
                            3.     //do something with foo
                            4. });
                            Parameter Type Usage Summary
                            config Object required

                            Properties

                            All properties are filtered from view.

                            _cache

                            _cache - private

                            Defined by havok/di/Di

                            Types: Array

                            An array of all objects created through the Di. If an object is requested more than once, it is retrieved from this array

                            _config

                            _config - private

                            Defined by havok/di/Di

                            Types: Object

                            The configuration object. Defines what to inject into objects

                            fallback

                            fallback

                            Defined by havok/di/Di

                            Types: havok/di/Di

                            Array of di instances. If no config is found for an indentifier, the fallback di instance will be used.
