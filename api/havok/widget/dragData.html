                            havok/widget/dragData

                            Hack to overcome dataTransfer.getData() in chrome

                            dataTransfer.getData() always returns a null in chrome in a dragover event therefore, this is effectively a global variable that is the drag data store. Use dragData along with the di container like this: require([&#39;get!./dragData&#39;], function(dragData){})
